# Analysis of US Tech Stock Returns and Risks

## Project Overview
This project conducts a comprehensive quantitative analysis of the return and risk characteristics of six leading U.S. technology stocks (Apple, Microsoft, Alphabet, Amazon, Tesla, Meta) from January 2021 to April 2025. The analysis covers core investment metrics including annualized return, annualized volatility, Sharpe ratio, maximum drawdown, and return correlation, providing data-driven investment insights.

---

## Key Visualizations

### 1. Cumulative Returns of US Tech Stocks (2021–2025)
This chart shows the growth of a $1 initial investment in each stock over the period, highlighting the overall performance trends.


---

### 2. Risk-Return Profile of US Tech Stocks
This scatter plot compares each stock’s annualized return against its annualized volatility, allowing for a direct assessment of risk-adjusted performance.


---

### 3. Return Correlation Heatmap
The heatmap illustrates the pairwise correlation of daily returns among the six stocks, useful for evaluating portfolio diversification potential.


---

### 4. Maximum Drawdown of US Tech Stocks
This chart displays the maximum peak-to-trough decline for each stock, showing their worst-case historical downside risk during the period.


---
